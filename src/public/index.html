Sobre o Projeto

Este conversor foi desenvolvido para facilitar a conversão de Bíblias em diferentes formatos XML para o padrão Zefenia XML, amplamente utilizado por aplicações bíblicas.

Tecnologias

Node.js, Express, XML2JS, Bootstrap
